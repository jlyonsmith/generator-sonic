## Service Component

Creates an new AngularJS service which can be used in the app. For further details refer to 
[AngularJS Docs](http://docs.angularjs.org/api/AUTO.$provide)



## Command Usage

***Example***

    $ yo sonic:service service1

This will create

    - app/
    	components/
    		service1/
        		|- service1-service.js
        		|- service1-service-test.js
