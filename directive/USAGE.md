## Directive Component

Creates an new AngularJS directive which can be used in the app. For further details refer to 
[AngularJS Docs](http://docs.angularjs.org/api/AUTO.$provide)

## Command Usage

***Example***

    $ yo sonic:directive name1

This will create

    - app/
    	components/
    		service1/
        		|- name1-directive.js
        		|- name1-directive-test.js
        		|- name1.tpl.html
        		|- name1.js
